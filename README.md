# MusicPen_Swift

## 개요
`YOLOv5` 와 `SwiftUI` 을 이용한 필기 인식 기반 악보 작성 앱으로, 2021년 한이음 ICT 공모전 입선, 2021년 홍익대학교 졸업 프로젝트 장려상 수상작입니다
